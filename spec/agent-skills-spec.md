# Agent Skills Specification

The full platform specification is available at:

**https://agentskills.io/specification**

## Quick Reference

Skills in this repo follow the standard agent skills format:
- Each skill lives in its own directory under `skills/`
- Each skill directory must contain a `SKILL.md` with valid YAML frontmatter
- The `name` field in frontmatter must match the directory name

See `template/SKILL.md` for the boilerplate.
